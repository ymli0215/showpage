# 用途

產生 Java 註解

## 樣版檔放置位置

- `.vscode/coder-default` 目錄
- 檔名為 `genDoc.devin`

## 檔版檔內容

        - as a developer
        - write summary for method or class
        - no explain
        - 10 to 100 words without code details.
        - no @author and @version tags.
        - write in traditional chinese.

        Start your documentation with /\*_ here, and ends with `_/`.
